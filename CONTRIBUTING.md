# Contributing to Lichess4Chess

Thank you for your interest in contributing to Lichess4Chess! 🎉
Whether you want to fix bugs, suggest features, or improve documentation — contributions of all kinds are welcome.

===========================================================
🛠 How to Contribute
===========================================================

1. Fork the repository.

   git fork https://github.com/ryrden/lichess4chess.git
   cd lichess4chess

2. Create a new branch.

   git checkout -b feature/your-feature-name

3. Make your changes and commit.

   git commit -m "Describe your change briefly"

4. Push your branch.

   git push origin feature/your-feature-name

5. Open a Pull Request on GitHub.

If you’re unsure how to do this, refer to:
https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request

===========================================================
🐛 Reporting Issues / Suggesting Features
===========================================================

- Open a GitHub Issue.
- Clearly describe:
  • What happened or what you suggest
  • Steps to reproduce (if applicable)
  • Screenshots/logs if useful

===========================================================
🧪 Running the Project Locally
===========================================================

See "How to run the project" in the README.

Before submitting your contribution:

✅ Ensure the project builds successfully  
✅ Test that the extension works in Chrome  
✅ Follow naming and folder structure conventions
